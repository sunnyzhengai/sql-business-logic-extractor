# September 2026 — AI Agent + Skills

## North star
Ship an AI agent (with composable skills) that orchestrates the 4 tools, and add it as a first-class offering on the website.

## Themes
1. **Agent** — multi-step orchestration over the 4 tools
2. **Skills** — modular capabilities the agent can call (extract columns, lineage, business logic, summarize, etc.)
3. **Website integration** — agent + skills as a new tier / product on the site

## Key deliverables (one focus per week)

- [ ] **Week 1 (Aug 31-Sep 6):** Build AI agent — architecture decided (Claude Agent SDK?), agent loop working end-to-end on a sample task
- [ ] **Week 2 (Sep 7-13):** Build skills — wrap each of the 4 tools as skills + add 2-3 net-new skills (e.g. anomaly explanation, schema diff, governance gap report)
- [ ] **Week 3 (Sep 14-20):** Add agent + skills to website — UI, pricing tier, demo video
- [ ] **Week 4 (Sep 21-27):** Write articles + reach out to customers — launch post on LinkedIn, follow-up emails to August customers, demo to Collibra contacts

## Risks / unknowns
- Inference cost per session — model choice + caching strategy matter for margins
- Skills sprawl — don't add a skill unless it earns its keep
- Agent failure modes in healthcare context: do NOT let it hallucinate clinical logic

## Out of scope
- Multi-agent / agent-to-agent (later)
- On-prem agent (model-hosting complexity — defer)
