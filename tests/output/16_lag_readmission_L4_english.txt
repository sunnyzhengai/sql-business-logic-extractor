prev_discharge
  Previous/next value comparison using PAT_ID, HOSP_DISCH_TIME, HOSP_ADMSN_TIME
  Technical: LAG(hosp_disch_time) OVER (PARTITION BY pat_id ORDER BY hosp_admsn_time)

days_since_last_discharge
  Previous/next value comparison using DAY, PAT_ID, HOSP_DISCH_TIME, HOSP_ADMSN_TIME
  Technical: DATEDIFF(day, LAG(hosp_disch_time) OVER (PARTITION BY pat_id ORDER BY hosp_admsn_time), HOSP_ADMSN_TIME)
