readmissions (aggregate)
  Resolved: SUM(CASE WHEN wr.days_to_readmit <= 30 THEN 1 ELSE 0 END)
  Base columns: .DAY, PAT_ENC_HSP.HOSP_DISCH_TIME, PAT_ENC_HSP.PAT_ID, PAT_ENC_HSP.HOSP_ADMSN_TIME
  Base tables: PAT_ENC_HSP
  Filters:
    - NOT e.HOSP_DISCH_TIME IS NULL
    - e.ADT_PAT_CLASS_C = 1
    - e.HOSP_ADMSN_TIME >= '2025-01-01'
    - COUNT(*) >= 50
  Chain:
    -> query.readmissions = SUM(f.is_30day_readmit) AS readmissions (aggregate)
      -> flagged.is_30day_readmit = CASE WHEN wr.days_to_readmit <= 30 THEN 1 ELSE 0 END AS is_30day_readmit (case)
        -> with_readmit.days_to_readmit = DATEDIFF(DAY, ie.HOSP_DISCH_TIME, LEAD(ie.HOSP_ADMSN_TIME) OVER (PARTITION BY ie.PAT_ID ORDER BY ie.HOSP_ADMSN_TIME)) AS days_to_readmit (window)
          -> index_encounters.HOSP_DISCH_TIME (passthrough)
            -> index_encounters.PAT_ID (passthrough)
              -> index_encounters.HOSP_ADMSN_TIME (passthrough)
                -> index_encounters.HOSP_ADMSN_TIME (passthrough)
