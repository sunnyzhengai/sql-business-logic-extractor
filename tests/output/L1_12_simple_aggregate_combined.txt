======================================================================
  L1_12_simple_aggregate
======================================================================

FORMATTED SQL:
----------------------------------------
SELECT
  dep.DEPARTMENT_NAME,
  COUNT(DISTINCT e.PAT_ENC_CSN_ID) AS encounter_count,
  AVG(DATEDIFF(DAY, e.HOSP_ADMSN_TIME, e.HOSP_DISCH_TIME)) AS avg_los
FROM PAT_ENC_HSP AS e
JOIN CLARITY_DEP AS dep
  ON e.DEPARTMENT_ID = dep.DEPARTMENT_ID
WHERE
  NOT e.HOSP_DISCH_TIME IS NULL
GROUP BY
  dep.DEPARTMENT_NAME

BUSINESS DEFINITIONS (Plain English):
----------------------------------------
  encounter_count
    Count of distinct Contact Serial Number — unique encounter identifier
    Source: PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Condition: Hospital discharge date and time is not null
    Technical: COUNT(DISTINCT pat_enc_hsp.pat_enc_csn_id)

  avg_los
    Average of DAY, Hospital admission date and time, Hospital discharge date and time
    Source: PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Condition: Hospital discharge date and time is not null
    Technical: AVG(DATEDIFF(day, pat_enc_hsp.hosp_admsn_time, pat_enc_hsp.hosp_disch_time))

  filter:where:PAT_ENC_HSP.HOSP_DISCH_TIME
    Filter: only include records where Hospital discharge date and time is not null
    Source: PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Technical: NOT pat_enc_hsp.hosp_disch_time IS NULL

LINEAGE (Resolved to base tables):
----------------------------------------
  DEPARTMENT_NAME (passthrough)
    Resolved: CLARITY_DEP.DEPARTMENT_NAME
    Base columns: CLARITY_DEP.DEPARTMENT_NAME
    Base tables: CLARITY_DEP
    Filter: NOT e.HOSP_DISCH_TIME IS NULL

  encounter_count (aggregate)
    Resolved: COUNT(DISTINCT e.PAT_ENC_CSN_ID)
    Base columns: PAT_ENC_HSP.PAT_ENC_CSN_ID
    Base tables: PAT_ENC_HSP
    Filter: NOT e.HOSP_DISCH_TIME IS NULL

  avg_los (aggregate)
    Resolved: AVG(DATEDIFF(DAY, e.HOSP_ADMSN_TIME, e.HOSP_DISCH_TIME))
    Base columns: PAT_ENC_HSP.DAY, PAT_ENC_HSP.HOSP_ADMSN_TIME, PAT_ENC_HSP.HOSP_DISCH_TIME
    Base tables: PAT_ENC_HSP
    Filter: NOT e.HOSP_DISCH_TIME IS NULL
