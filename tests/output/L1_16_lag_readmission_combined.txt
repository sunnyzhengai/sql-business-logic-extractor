======================================================================
  L1_16_lag_readmission
======================================================================

FORMATTED SQL:
----------------------------------------
SELECT
  PAT_ID,
  PAT_ENC_CSN_ID,
  HOSP_ADMSN_TIME,
  HOSP_DISCH_TIME,
  LAG(HOSP_DISCH_TIME) OVER (PARTITION BY PAT_ID ORDER BY HOSP_ADMSN_TIME) AS prev_discharge,
  DATEDIFF(
    DAY,
    LAG(HOSP_DISCH_TIME) OVER (PARTITION BY PAT_ID ORDER BY HOSP_ADMSN_TIME),
    HOSP_ADMSN_TIME
  ) AS days_since_last_discharge
FROM PAT_ENC_HSP
WHERE
  NOT HOSP_DISCH_TIME IS NULL

BUSINESS DEFINITIONS (Plain English):
----------------------------------------
  prev_discharge
    Previous/next value comparison using PAT_ID, HOSP_DISCH_TIME, HOSP_ADMSN_TIME
    Technical: LAG(hosp_disch_time) OVER (PARTITION BY pat_id ORDER BY hosp_admsn_time)

  days_since_last_discharge
    Previous/next value comparison using DAY, PAT_ID, HOSP_DISCH_TIME, HOSP_ADMSN_TIME
    Technical: DATEDIFF(day, LAG(hosp_disch_time) OVER (PARTITION BY pat_id ORDER BY hosp_admsn_time), HOSP_ADMSN_TIME)

  filter:where:HOSP_DISCH_TIME
    Filter: only include records where hosp_disch_time is not null
    Technical: NOT hosp_disch_time IS NULL
