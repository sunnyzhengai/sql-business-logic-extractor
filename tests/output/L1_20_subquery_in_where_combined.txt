  L1_20_subquery_in_where
======================================================================

FORMATTED SQL:
----------------------------------------
SELECT
  p.PAT_MRN_ID,
  p.PAT_FIRST_NAME,
  p.PAT_LAST_NAME
FROM PATIENT AS p
WHERE
  p.PAT_ID IN (
    SELECT DISTINCT
      e.PAT_ID
    FROM PAT_ENC_HSP AS e
    JOIN HSP_ACCT_DX_LIST AS dl
      ON e.HSP_ACCOUNT_ID = dl.HSP_ACCOUNT_ID
    JOIN CLARITY_EDG AS dx
      ON dl.DX_ID = dx.DX_ID
    WHERE
      dx.ICD10_CODE LIKE 'I50%'
  )

BUSINESS DEFINITIONS (Plain English):
----------------------------------------

    Source: CLARITY_EDG (Diagnosis reference table (ICD-10, descriptions)), HSP_ACCT_DX_LIST (Diagnosis list for hospital accounts (primary, secondary, etc.)), PATIENT (Master patient demographics table), PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Technical: patient.pat_id IN (SELECT DISTINCT pat_enc_hsp.pat_id FROM pat_enc_hsp AS e JOIN hsp_acct_dx_list AS dl ON pat_enc_hsp.hsp_account_id = hsp_acct_dx_list.hsp_account_id JOIN clarity_edg AS dx ON hsp_acct_dx_list.dx_id = clarity_edg.dx_id WHERE clarity_edg.icd10_code LIKE 'I50%')

  filter:where:CLARITY_EDG.ICD10_CODE
    Filter: only include records where ICD-10 diagnosis code matches pattern 'I50%'
    Source: CLARITY_EDG (Diagnosis reference table (ICD-10, descriptions))
