FORMATTED SQL:
----------------------------------------
SELECT
  e.PAT_ENC_CSN_ID,
  (
    SELECT
      MAX(fm.RECORDED_TIME)
    FROM IP_FLWSHT_MEAS AS fm
    WHERE
      fm.PAT_ENC_CSN_ID = e.PAT_ENC_CSN_ID AND fm.FLO_MEAS_ID = '5'
  ) AS last_bp_time
FROM PAT_ENC_HSP AS e

BUSINESS DEFINITIONS (Plain English):
----------------------------------------
  last_bp_time
    Value derived from a sub-query
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments)), PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Condition: Encounter identifier equals pat_enc_hsp.pat_enc_csn_id
    Condition: Flowsheet measurement type (5=Blood Pressure, 8=Heart Rate, etc.) equals '5'
    Technical: (SELECT MAX(ip_flwsht_meas.recorded_time) FROM ip_flwsht_meas AS fm WHERE ip_flwsht_meas.pat_enc_csn_id = pat_enc_hsp.pat_enc_csn_id AND ip_flwsht_meas.flo_meas_id = '5')

  filter:where:IP_FLWSHT_MEAS.PAT_ENC_CSN_ID
    Filter: only include records where Encounter identifier equals pat_enc_hsp.pat_enc_csn_id
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments)), PAT_ENC_HSP (Hospital patient encounters (inpatient, ED, observation))
    Technical: ip_flwsht_meas.pat_enc_csn_id = pat_enc_hsp.pat_enc_csn_id

  filter:where:IP_FLWSHT_MEAS.FLO_MEAS_ID
    Filter: only include records where Flowsheet measurement type (5=Blood Pressure, 8=Heart Rate, etc.) equals '5'
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments))
    Technical: ip_flwsht_meas.flo_meas_id = '5'

  MAX(fm.RECORDED_TIME)
    Maximum of Time the measurement was recorded
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments))
    Condition: Encounter identifier equals e.pat_enc_csn_id
    Condition: Flowsheet measurement type (5=Blood Pressure, 8=Heart Rate, etc.) equals '5'
    Technical: MAX(ip_flwsht_meas.recorded_time)

  filter:where:IP_FLWSHT_MEAS.PAT_ENC_CSN_ID
    Filter: only include records where Encounter identifier equals e.pat_enc_csn_id
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments)), e
    Technical: ip_flwsht_meas.pat_enc_csn_id = e.pat_enc_csn_id

  filter:where:IP_FLWSHT_MEAS.FLO_MEAS_ID
    Filter: only include records where Flowsheet measurement type (5=Blood Pressure, 8=Heart Rate, etc.) equals '5'
    Source: IP_FLWSHT_MEAS (Inpatient flowsheet measurements (vitals, assessments))
    Technical: ip_flwsht_meas.flo_meas_id = '5'
