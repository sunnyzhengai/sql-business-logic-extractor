FORMATTED SQL:
----------------------------------------
SELECT
  CASE
    WHEN TOTAL_CHARGES <= 5000
    THEN 'Low'
    WHEN TOTAL_CHARGES <= 20000
    THEN 'Medium'
    ELSE 'High'
  END AS cost_category
FROM HSP_TRANSACTIONS

BUSINESS DEFINITIONS (Plain English):
----------------------------------------
  cost_category
    Classified as:
  - 'Low' when total_charges is at most 5000
  - 'Medium' when total_charges is at most 20000
  - 'High' otherwise
    Technical: CASE WHEN total_charges <= 5000 THEN 'Low' WHEN total_charges <= 20000 THEN 'Medium' ELSE 'High' END
